# 여러 채널에 디지털 참여 만들기

## Salesforce의 디지털 참여 도구

 - 여러 채널—전화, 이메일, 문자 메시지, SNS 등—을 추가하는 것이 전체적인 Service Cloud 설정 프로세스의 두 번째 단계

 - 기본 채널인 이메일과 웹 사이트 양식을 만들어 고객들이 어디서든지 원하는 채널에서 사례를 접수하여 서비스를 받을 수 있도록 보다 나은 서비스를 제공해준다
