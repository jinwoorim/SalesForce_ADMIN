# 앱 홈 번개 페이지 구축

## 앱 홈 번개 페이지

 - Salesforce 모바일 앱 및 Lightning Experience 앱 탐색 모음에 앱에 대한 사용자 지정 홈 페이지를 추가하여 사용자가 해당 앱에서 가장 중요한 객체 및 항목에 쉽게 액세스할 수 있도록 한다
