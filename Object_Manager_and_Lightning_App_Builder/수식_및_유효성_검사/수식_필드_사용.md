# 수식 필드 사용

## 수식 필드란

 - 수많은 데이터 표시 방식을 관리 할 수 있는 강력한 도구
    - 레코드에서 두개의 숫자 필드를 가져와 계산
    - 레코드의 페이지 레이아웃에서 중요한 정보에 쉽게 액세스 할 수 있도록 필드를 클릭 가능한 하이퍼링크로 바꿀수있다
    - 두 개의 날짜를 가져와서 두 날짜 간 일수를 계산
