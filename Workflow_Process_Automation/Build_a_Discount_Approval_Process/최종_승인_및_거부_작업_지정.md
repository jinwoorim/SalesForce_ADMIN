# 최종 승인 및 거부 작업 지정

 - 승인 프로세스의 마지막 단계는 최종 승인 조치와 최종 거부 조치를 만드는 것이다

 - 최종 승인 작업은 필요한 모든 승인을 받을 때 발생한다. 최종 거부 작업은 승인자가 요청을 거부하고 최종 거부 상태로 이동할 때 발생한다
